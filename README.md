# Josh Benner's Dotfiles

Dotfiles managed with [Ansible](http://ansible.com).

## Install Ansible

### Ubuntu

```sh
$ sudo apt-get install ansible
```

### Arch

```sh
$ sudo pacman -S ansible
```

### macOS

```sh
$ brew install ansible
```

## Run Ansible

```sh
cd ~/dotfiles
./run <work or personal>
```
